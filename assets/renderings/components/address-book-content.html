Activating a User to Your Account.

Activating a user to your account

As a site administrator or support person with access to the Traveler Profiles page, you can add a user to create a traveler profile. You add the initial profile information, and the user enters the rest of the profile information after receiving the activation email. Once the user has activated their account they will then be a part of your address book and can then be invited/added to any event site that you have created.

THINGS TO REMEMBER:

* Once added to your account you will still need to assign/invite them to any event site you have created that you have set to private "login only"

* When you activate a user, the process sends out an activation email with a log-in link to the user as an initial invitation

Traveler Profile

The Traveler profile contains personal identification needed to book services. It also allows you to store preferences for services and communications. Using information in your profile reduces the time it takes you to arrange for services, because you don't have to enter them each time you purchase a flight. It also expedites when dealing with any post ticketing support requests.
